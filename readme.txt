Easily Manage Team Members Detail On Your WordPress Blog

== Description ==

A Plugin to add staff member's or employee's of the company/organisation on your wordpress blog. Simply unzip and upload appointment-calendar directory to `/wp-content/plugins/` directory and activate the plugin.

Use shortcode [addteammember] to insert calendar into any post or page.

Thats it, you are done.
